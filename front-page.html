Digit Triplet Test


Please make sure you are in a quiet environment and wearing headphones before starting the test.
( Ka tswee-tswee, dira gore omo lefong lele senang modumo le gore o dirisa di headphones pele o simolola teko e. )

After pressing the start button, this test will play three digits, with some background noise.
( Morago ga gago tobetsa gao simollang teng, teko ye e tlaa tshameka dinomoro di le tharo go na le modumo ko morago ga tsone. )
